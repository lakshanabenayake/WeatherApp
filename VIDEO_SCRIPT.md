## Troubleshooting

### If recording exceeds 60 seconds:
- Speed up your narration slightly
- Cut out pauses between actions
- Use video editing software to trim dead time
- Skip the introduction, go straight to demonstration

### If app doesn't show cached data offline:
- Make sure you successfully fetched weather at least once before testing offline
- Check that the error message appears (this confirms offline state)
- The cached data should appear automatically below the error

### If request URL is not visible:
- Scroll down in the app to reveal it
- It's at the very bottom in a small, monospace font
- Zoom in slightly if needed for recording
